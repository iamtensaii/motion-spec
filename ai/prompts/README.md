# AI Prompt Templates

Copy-paste these into your AI tool's config to make it generate motion-spec animations automatically.

| File | Tool | Where to paste |
|------|------|---------------|
| [claude-code.md](./claude-code.md) | Claude Code | Your project's `CLAUDE.md` |
| [cursor.md](./cursor.md) | Cursor | `.cursorrules` in project root |
| [v0.md](./v0.md) | Vercel v0 | Prepend to your v0 prompt |

For any other LLM, reference the machine-readable spec directly:
```
https://raw.githubusercontent.com/iamtensaii/motion-spec/main/ai/llms.txt
```
