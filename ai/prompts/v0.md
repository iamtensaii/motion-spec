# motion-spec prompt for Vercel v0

Add to your v0 prompt:

```
Use motion-spec for all animations (https://motion-spec.dev).

Available components:
  <Entrance intent="default|focus|hero|subtle|immediate">
  <Exit intent="default|dismiss|replace|immediate">
  <Feedback intent="success|error|warning|loading|confirm">
  <Stagger intent="list|grid|cascade|random">
  <NavigationTransition intent="forward|back|lateral|modal|dismiss">
  <Celebration intent="milestone|reward|subtle">

All components from: import { ... } from "@motion-spec/react"

Rules:
- Dialogs/modals → <Entrance intent="focus">
- Success states → <Feedback intent="success">
- Lists/feeds → <Stagger intent="list">
- Page navigation → <NavigationTransition intent="forward">
- Achievements → <Celebration intent="milestone">
- Never use raw transition: or animation: CSS for semantic UI motion
```
